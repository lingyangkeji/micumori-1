# お見積り作成

日本語見積り作成ツール。Word化、PPT化、Excell化、TM作成、メモリ作成の見積りを簡単に作成できます。

## ローカルで実行する方法

### 前提条件

- **Node.js** v18 以上（https://nodejs.org/ からダウンロード）
- **pnpm**（以下のコマンドでインストール）

```bash
npm install -g pnpm
```

### 手順

#### 1. 依存パッケージをインストール

```bash
pnpm install
```

#### 2. 開発サーバーを起動

```bash
pnpm dev:local
```

#### 3. ブラウザでアクセス

```
http://localhost:5000
```

### 本番ビルド

```bash
pnpm build:local
pnpm start:local
```

## 機能一覧

- 顧客名入力
- 5つの項目（Word化、PPT化、Excell化、TM作成、メモリ作成）から複数選択可能
- 各項目のサブ項目を複数選択可能
- 単価は標準価格が設定済み、変更も可能
- カスタム数字キーボードで数量入力
- 小計・消費税（10%）・税込合計を自動計算
- 見積りテキストを出力・コピー可能
